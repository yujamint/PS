# 후위식 연산(postfix)

## 문제
```
[설명]
후위연산식이 주어지면 연산한 결과를 출력하는 프로그램을 작성하세요.

만약 3*(5+2)-9 을 후위연산식으로 표현하면 352+*9- 로 표현되며 그 결과는 12입니다.


[입력]
첫 줄에 후위연산식이 주어집니다. 연산식의 길이는 50을 넘지 않습니다.

식은 1~9의 숫자와 +, -, *, / 연산자로만 이루어진다.


[출력]
연산한 결과를 출력합니다.
```
```
예시 입력 1 
352+*9-

예시 출력 1
12
```

**문제 풀이방법**
- 입력받은 문자열을 for문을 통해 문자 단위로 다음과 같이 처리한다.
    - 숫자일 경우(isDigit 함수로 구분) stack에 push한다.
    - 기호일 경우, stack에서 2개의 숫자를 꺼낸 후 기호에 맞게 연산한다.
    
**얻어갈 것**
- 기호 처리하는 과정에서 방향을 잡지 못 했다. '+', '-' , '*', '/' 4개의 연산자밖에 없는데 그냥 if 문으로 처리하면 될 것을 너무 오래 고민했다.
- char형 -> int형 변환 과정(ASCII CODE) 다시 한 번 숙지!
